## Critical Findings

### 1. ⚠️ [MEDIUM] Retry Logic Missing Backoff

**File**: `src/main/java/com/nexusfin/equity/service/impl/TechPlatformUserClientImpl.java:51-78`

**Problem**:
```java
for (int attempt = 1; attempt <= maxAttempts; attempt++) {
    try {
        // ... HTTP request ...
    } catch (RestClientException exception) {
        // Immediate retry — no backoff, no jitter
        if (attempt == maxAttempts) {
            throw new BizException(...);
        }
    }
}
```

当上游服务暂时不可用时，连续快速重试可能加重其负载，也增加本端延迟。

**Recommendation**:
```java
catch (RestClientException exception) {
    if (attempt < maxAttempts) {
        try {
            Thread.sleep(100L * attempt);  // 100ms, 200ms, ...
        } catch (InterruptedException e) {
            Thread.currentThread().interrupt();
            throw new BizException("UPSTREAM_AUTH_FAILED", "Retry interrupted");
        }
    } else {
        throw new BizException("UPSTREAM_AUTH_FAILED", "Failed to verify tech platform token");
    }
}
```

**Severity**: Medium (可观测性和友好性，非功能性缺陷)
**Action**: AUTO-FIX
