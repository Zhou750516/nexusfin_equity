2. **该写什么**：状态、阻塞、决定。**不该写什么**：每日 todo（去 `<date>.md`）、详细方案（去 `topics/<主题>/`）。
3. **占位符（` < ... > `）必须替换**——出现在最终状态里说明这一行没维护。
4. **历史快照不要删**——已收口的事项用 ✅ 标记，迁移到 §5，永不删除（防止 “以为做过了实际没做”）。
