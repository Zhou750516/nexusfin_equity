# 小花联合登录核心入口（本地骨架） Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为艾博生 H5 增加一个可本地联调的“小花联合登录核心入口骨架”，打通“外部跳转参数承接 → 艾博生后端建立本地登录态 → H5 场景分流”的最小闭环。

**Architecture:** 后端新增公开的 `/api/auth/joint-login` 入口，接收 `token + scene + 订单上下文`，通过 `XiaohuaAuthClient` 抽象获取外部用户身份，再用现有 `JwtUtil + CookieUtil + member_info/member_channel` 体系建立本地登录态。前端新增 `/joint-entry` 承接页和场景占位页，只负责调用本地后端、处理页面态和路由跳转，不直接调用小花后端。

**Tech Stack:** Java 17, Spring Boot 3.2, MockMvc, React 19, TypeScript 5.6, axios, wouter, Vitest

```java
@RestController
public class JointLoginController {

    private final JointLoginService jointLoginService;
    private final CookieUtil cookieUtil;

    public JointLoginController(JointLoginService jointLoginService, CookieUtil cookieUtil) {
        this.jointLoginService = jointLoginService;
        this.cookieUtil = cookieUtil;
    }

    @PostMapping("/api/auth/joint-login")
    public ResponseEntity<Result<JointLoginResponse>> jointLogin(
            @Valid @RequestBody JointLoginRequest request
    ) {
        JointLoginService.JointLoginResult result = jointLoginService.login(request);
        JointLoginResponse response = new JointLoginResponse(
                true,
                result.scene(),
                result.targetPage(),
                result.benefitOrderNo(),
                result.localUserReady()
        );
        return ResponseEntity.ok()
                .header(HttpHeaders.SET_COOKIE, cookieUtil.buildAuthCookie(result.jwtToken()).toString())
                .body(Result.success(response));
    }
}
```

```java
@Service
public class JointLoginServiceImpl implements JointLoginService {

    private final XiaohuaAuthClient xiaohuaAuthClient;
    private final MemberInfoRepository memberInfoRepository;
    private final MemberChannelRepository memberChannelRepository;
    private final JwtUtil jwtUtil;
    private final SensitiveDataCipher sensitiveDataCipher;
    private final JointLoginTargetPageResolver targetPageResolver;

    @Override
    @Transactional
    public JointLoginResult login(JointLoginRequest request) {
        XiaohuaAuthClient.JointIdentity identity = xiaohuaAuthClient.exchange(request.token());
        MemberInfo memberInfo = findOrCreateMember(identity);
        ensureChannelBinding(memberInfo, identity.externalUserId(), identity.channelCode());
        String jwtToken = jwtUtil.generateToken(memberInfo.getMemberId(), identity.externalUserId());
        return new JointLoginResult(
                jwtToken,
                request.scene(),
                targetPageResolver.resolve(request.scene()),
                request.benefitOrderNo(),
                true
        );
    }
}
```

```ts
export function resolveJointEntryTarget(result: JointLoginResult) {
  const query = new URLSearchParams();
  query.set("scene", result.scene);
  if (result.benefitOrderNo) {
    query.set("benefitOrderNo", result.benefitOrderNo);
  }
  if (result.targetPage === "joint-refund-entry") {
    return `/joint-refund-entry?${query.toString()}`;
  }
  return `/joint-dispatch?${query.toString()}`;
}
```

```tsx
export default function JointEntryPage() {
  const [, navigate] = useLocation();
  const { t } = useI18n();

  useEffect(() => {
    async function run() {
      const search = new URLSearchParams(window.location.search);
      const token = search.get("token");
      const scene = search.get("scene");
      if (!token || !scene) {
        return;
      }
      const result = await jointLogin({ token, scene: scene as JointLoginParams["scene"] });
      navigate(resolveJointEntryTarget(result));
    }
    void run();
  }, [navigate]);

  return <PageFeedback title={t("jointEntry.loadingTitle")} description={t("jointEntry.loadingDescription")} />;
}
```
