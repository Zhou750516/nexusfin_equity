# 2026-04-21 Playwright H5 多语言冒烟报告

## 1. 验证目标

本轮仅覆盖 H5 首页 `/` 的多语言冒烟验证，不覆盖完整业务主流程。

验证范围：

- 浏览器语言自动匹配：
  - `zh-CN`
  - `zh-TW`
  - `en-US`
  - `vi-VN`
- 页面手动切换语言：
  - `简中`
  - `繁中`
  - `EN`
  - `VI`
- 页面关键文案切换正确性

## 2. 验证环境

- H5 地址：`http://127.0.0.1:3000/`
- 后端地址：`http://127.0.0.1:8080`
- 验证时间：`2026-04-21`
- 验证方式：Playwright Headless 自动化验证
- 原始日志：`test/artifacts/20260421_playwright_h5_i18n_smoke.json:1`

## 3. 验证口径

本轮对每种语言统一检查以下内容：

1. 页面语言标题是否切换正确；
2. “借款金额”文案是否切换正确；
3. “借多久 / Loan Term / Thời hạn vay”文案是否切换正确；
4. “确认借款 / Confirm Loan / Xác nhận vay”按钮文案是否切换正确；
5. 页面语言选择器当前值是否正确；
6. `document.documentElement.lang` 是否同步更新；
7. `localStorage` 中的 `nexusfin.h5.locale` 是否同步更新。
